## Getting Started

This is an example of how you can run PWeb+

### Installation

- For Linux & Termux :
```
git clone https://github.com/P1shi/PWeb-Plus.git
cd PWeb-Plus
bash run.sh
```

- For Windows:
```
git clone https://github.com/P1shi/PWeb-Plus.git
cd PWeb-Plus
start run.bat
``````
